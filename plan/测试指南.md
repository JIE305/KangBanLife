## 一、前端测试（Vitest）

### 现有测试

`KangBan-vue3/tests/stores/` 下有 2 个测试文件：

- `health.test.js` — 健康 Store（饮水/BMI/清单）
- `user.test.js` — 用户 Store（登录/注册/登出/持久化）

### 修复测试脚本

`package.json` 当前缺少 `test` 脚本，需添加：

```json
{
  "scripts": {
    "dev": "vite",
    "build": "vite build",
    "preview": "vite preview",
    "test": "vitest run",
    "test:watch": "vitest"
  }
}
```

### 运行

```bash
cd KangBan-vue3
npm test              # 一次性运行
npm run test:watch    # 监听模式
```

### conftest.py 示例

```python
import pytest
import os
import tempfile
from app import create_app

@pytest.fixture
def app():
    # 使用临时数据库
    db_fd, db_path = tempfile.mkstemp(suffix='.db')
    app = create_app()
    app.config['DATABASE'] = db_path
    app.config['TESTING'] = True
    app.config['JWT_SECRET_KEY'] = 'test-secret'

    with app.app_context():
        from utils import get_db
        db = get_db()
        with open('schema_lite.sql', 'r', encoding='utf-8') as f:
            db.executescript(f.read())
        db.commit()

    yield app

    os.close(db_fd)
    os.unlink(db_path)

@pytest.fixture
def client(app):
    return app.test_client()

@pytest.fixture
def auth_token(client):
    """注册用户并返回 token"""
    client.post('/api/v1/auth/register', json={
        'studentId': 'test001',
        'username': '测试',
        'phone': '13800001111',
        'password': '123456'
    })
    resp = client.post('/api/v1/auth/login', json={
        'studentId': 'test001',
        'password': '123456'
    })
    return resp.get_json()['data']['token']
```

### 测试用例示例（test_auth.py）

```python
def test_login_success(client):
    resp = client.post('/api/v1/auth/register', json={
        'studentId': '2024001',
        'username': '张三',
        'phone': '13800001111',
        'password': '123456'
    })
    assert resp.status_code == 201

    resp = client.post('/api/v1/auth/login', json={
        'studentId': '2024001',
        'password': '123456'
    })
    assert resp.status_code == 200
    data = resp.get_json()
    assert 'token' in data['data']
    assert data['data']['user']['student_id'] == '2024001'

def test_login_wrong_password(client):
    resp = client.post('/api/v1/auth/login', json={
        'studentId': '2024001',
        'password': 'wrong'
    })
    assert resp.status_code == 401

def test_duplicate_register(client):
    payload = {'studentId': 'dup1', 'username': 'duplicate', 'phone': '13800001111', 'password': '123456'}
    client.post('/api/v1/auth/register', json=payload)
    resp = client.post('/api/v1/auth/register', json=payload)
    assert resp.status_code == 409
```

### 运行

```bash
cd KangBan-backend
pytest tests/ -v           # 详细输出
pytest tests/ --cov=api    # 覆盖率（需 pip install pytest-cov）
```

## 三、API 端点手动测试

### 使用 curl

```bash
# 登录
TOKEN=$(curl -s -X POST http://localhost:5000/api/v1/auth/login \
  -H "Content-Type: application/json" \
  -d '{"studentId":"2024001","password":"123456"}' | jq -r '.data.token')

# 带 token 的请求
curl -H "Authorization: Bearer $TOKEN" http://localhost:5000/api/v1/user/profile
```

### 使用 VS Code REST Client 插件

创建 `tests/api.http`：

